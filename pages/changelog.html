v1.4 - July 23, 2021

Major:

  • Multi mode "Share" button
  • Cherry and Rainforest themes

Minor/QoL:

  • Moved watermark to the left of foals

v1.3 - July 11, 2021

Major:

  • Added a new page for merging multiple different horses together ("Multi Mode")

Minor/QoL:

  • Changed favicon from Horse Reality's logo to the R
  • Added an optional dark mode
  • Added proper error pages (try it out!)
  • Added a settings menu to reduce button clutter
  • Added a navigation dropup
  • Fixed weird escaping on some characters in horse names
  • Changed minimum lifenumber length to 1

Multi Mode notes:

  • Select layers from top to bottom for them to display in the correct order
  • You can add a maximum of 6 horses at a time
  • To get an individual layer PNG, check the box of the layer you want. Right-click or long-press the preview image to open the layer's URL.

v1.2.1 - July 3, 2021

  • Added Horse Reality watermark to all merged images for additional attribution.

v1.2 - June 8, 2021

  • Fixed edge case where sometimes some layers were opened in LA mode instead of RGBA, causing the tool to fail.
  • Moved both footer boxes to the left side so that they would not overlap on mobile devices.

v1.1 - June 6, 2021

  • Added support for horse pages with both a foal and a mare.

v1.0 - May 20, 2021

  • Initial stable build.
